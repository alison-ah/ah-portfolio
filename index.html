Description

Full Stack Collection App
Family library card catalog

  • Application of accessible design practices
  • Updating and adapting legacy code, leveraging AI assistance for learning and troubleshooting

Tools

  • Replit
  • Chrome DevTools
  • Accessibility, HTML, and CSS validators

Description

The Family Library app allows you to add, edit, and delete books with a REST API so you can organize your collection with a basic “card catalog.” You can also search by title or author. It was built and deployed on Replit. It is an iteration on the female supervillain trading card app and recipe app Skillcrush projects.

During the COVID lock down, I organized our family’s book collection with a library-like format. This app can take that project to another level.
